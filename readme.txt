Transform your Ultimate Member static forms to Ajax.

== Description ==

It transform the login, register forms to ajax.

It partially works with the account update (validates only through ajax) in ultimate member, due to some restrictions on ultimate member plugin.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. The plugin is functional with one form, a license is required to unlock more than one form.
